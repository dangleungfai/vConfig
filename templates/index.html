    数据库备份内容
    
        
    当前备份将包含以下数据（仅数据库，不含配置文件等）：
    
        
      
            
    • 设备：设备列表（IP、主机名、类型、账号、连接方式等）
      • 
            
    • 备份日志：每次备份的执行记录（状态、耗时、配置文件路径等）
      • 
            
    • 备份任务记录：全量/单台备份任务及进度
      • 
            
    • 应用设置：系统名称、时区、备份频率、LDAP、Logo 等键值对
      • 
            
    • 登录日志：用户登录记录（用户名、IP、认证方式）
      • 
            
    • 审计日志：敏感操作记录
      • 
            
    • 配置下发日志：配置推送记录
      • 
            
    • 用户：本地与 LDAP 用户及角色
      • 
            
    • 配置变动记录：首页配置变动卡片数据
      • 
            
    • 设备类型配置：设备类型及备份/连接命令配置

    自动发现
    
                    
    
                        通过 SNMP 从指定 IP 段自动发现网络设备（获取主机名、设备类型和管理 IP），并自动加入设备管理列表。
                    
    
                    
    
                        
                        
    
                            
                            用于控制后台定时执行自动发现规则的频率；建议在规则配置完善后再开启。
                        
    
                        
    
                            
                            
                        
    
                    
    
                    
    
                        
                        
    
                            
                            
                            
                            ms
                            
                            次
                        
    
                        用于自动发现时的 SNMP 访问参数；Community 可在单条规则中单独覆盖。超时为每次 SNMP 请求的等待时间，重试次数为失败后额外请求的次数。
                    
    

                    
    
                        
                        
    
                        
    
                            
                        
    
                        
                            每行：左侧选择设备类型（从设备类型管理同步），右侧输入匹配关键字（逗号分隔）。当 SNMP 返回值包含任一关键字时，归类为该设备类型。
                        
                    
    

                    
    
                        
                        
    
                            按分隔符
                            
                            分段，取前
                            
                            段拼成主机名
                        
    
                        主机名为域名格式时，按分隔符分段后取「前 N 段」作为主机名：无后缀如 ABC1.PE1. 取前 2 段得 ABC1.PE1；有后缀如 ABC1.PE1.example.com 取前 2 段得 ABC1.PE1。留空分隔符则不处理。
                    
    
                    
    
                        
                        
    
                            
                        
    
                        自动发现时，若设备列表中已存在相同主机名或相同 IP（依此处选择），则跳过不重复添加。默认按主机名。
                    
    
                    
    
                        
                        
    输入 IP 段或列表，立即扫描 SNMP 可达设备，仅展示结果，不会自动入库。
    
                        
    
                            
                            
                        
    
                        
    
                            
    
                        
    
                    
    

                    
    
                        
                        
    配置规则后，可一键运行按 IP 段 + SNMP 自动将设备加入「设备管理」。

    备份设置
    
                    
    
                        
                            
                        修改后需重启服务方可完全生效

                        如默认 SSH 失败时自动改用 Telnet；默认 Telnet 失败时自动改用 SSH，若都失败则按失败记录。
